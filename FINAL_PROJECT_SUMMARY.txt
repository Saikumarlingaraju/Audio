TASK 3: WORD VS SENTENCE LEVEL COMPARISON ✅ COMPLETED
==============================================================================

OBJECTIVE:
Compare accent classification performance between word-level segments and
full sentence-level utterances to understand the importance of temporal
context and prosodic information.

CONFIGURATION:
  Model: HuBERT Layer 3 (best from T2 analysis)
  Feature Dimensions: 768 (mean pooling)
  Classifier: MLP [768 → 256 → 128 → 6]
  Training: 20 epochs, Adam optimizer (lr=0.001)

DATASET STATISTICS:

Sentence-Level:
  - Total samples: 2798 full utterances
  - Train: 1958, Val: 420, Test: 420
  - Audio duration: ~3-5 seconds per utterance
  - Context: Full prosodic and rhythmic patterns

Word-Level:
  - Total samples: 5358 word segments (extracted via VAD)
  - Train: 3838, Val: 774, Test: 746
  - Audio duration: ~0.5-1 second per word
  - Context: Isolated words without prosody
  - Extraction: 3-10 words per original utterance

DETAILED RESULTS:

┌──────────────────┬──────────────┬──────────────┬──────────────┐
│ Level            │ Train Acc    │ Val Acc      │ Test Acc     │
├──────────────────┼──────────────┼──────────────┼──────────────┤
│ Sentence-Level   │   99.64%     │  100.00%     │   99.52%     │
│ Word-Level       │   96.46%     │   96.38%     │   94.77%     │
├──────────────────┼──────────────┼──────────────┼──────────────┤
│ Difference       │   +3.18%     │   +3.62%     │   +4.75%     │
└──────────────────┴──────────────┴──────────────┴──────────────┘

WINNER: Sentence-Level by 4.75% test accuracy

KEY FINDINGS:

1. CONVERGENCE SPEED:
   ✓ Sentence-level: Very fast (>99% by epoch 2)
   ✓ Word-level: Slower (reaches 94% by epoch 7)
   → Sentence-level features are more discriminative

2. TRAINING STABILITY:
   ✓ Sentence-level: Stable (val acc 99-100% throughout)
   ✓ Word-level: More volatile (val acc varies 90-96%)
   → Sentence-level provides consistent learning signal

3. GENERALIZATION:
   ✓ Sentence-level: Minimal gap (100% val → 99.52% test, 0.48% gap)
   ✓ Word-level: Larger gap (96.38% val → 94.77% test, 1.61% gap)
   → Sentence-level generalizes better

4. DATA EFFICIENCY:
   ✓ Sentence-level: 2798 samples → 99.52% accuracy
   ✓ Word-level: 5358 samples (1.9x more) → 94.77% accuracy
   → Sentence-level is more data-efficient

WHY SENTENCE-LEVEL WINS:

1. Prosodic Information:
   - Intonation patterns, rhythm, and stress patterns preserved
   - These suprasegmental features are strong accent indicators
   - Word-level segments lose critical prosodic context

2. Temporal Context:
   - Longer audio provides more temporal context
   - HuBERT captures longer-range dependencies
   - Words are too short to exhibit full accent characteristics

3. Coarticulation Effects:
   - Sentences show how sounds blend across word boundaries
   - Cross-word effects reveal accent-specific articulation
   - Isolated words miss natural speech patterns

4. Feature Richness:
   - 3-5 seconds of speech → richer feature representation
   - More frames → more robust statistics after pooling
   - Short segments may contain noise or incomplete phonemes

5. Accent Manifestation:
   - Accents most apparent in connected speech
   - Sentence-level captures speaking style and fluency
   - Word-level shows only phonetic variations
