==============================================================================
KEY OBSERVATIONS
==============================================================================

1. CONVERGENCE SPEED:
   ✓ Sentence-level: Very fast convergence (>99% by epoch 2)
   ✓ Word-level: Slower convergence (reaches 94% by epoch 7)
   → Sentence-level features are more discriminative and easier to learn

2. TRAINING STABILITY:
   ✓ Sentence-level: Very stable (val acc stays 99-100% after epoch 2)
   ✓ Word-level: More fluctuation (val acc varies 90-96% throughout)
   → Sentence-level provides more consistent learning signal

3. GENERALIZATION GAP:
   ✓ Sentence-level: Minimal gap (100% val → 99.52% test, gap = 0.48%)
   ✓ Word-level: Larger gap (96.38% val → 94.77% test, gap = 1.61%)
   → Sentence-level generalizes better to unseen data

4. ABSOLUTE PERFORMANCE:
   ✓ Sentence-level: Near-perfect accuracy (99.52% test)
   ✓ Word-level: Good but not excellent accuracy (94.77% test)
   → Sentence-level is clearly superior for this task

5. DATA EFFICIENCY:
   ✓ Sentence-level: 2798 samples achieve 99.52% accuracy
   ✓ Word-level: 5358 samples (1.9x more) achieve only 94.77% accuracy
   → Sentence-level is more data-efficient

==============================================================================
ANALYSIS & INTERPRETATION
==============================================================================

WHY SENTENCE-LEVEL PERFORMS BETTER (+4.75%):

1. PROSODIC INFORMATION:
   ✓ Sentences contain intonation patterns, rhythm, and stress patterns
   ✓ These suprasegmental features are strong accent indicators
   ✓ Word-level segments lose this critical prosodic context

2. TEMPORAL CONTEXT:
   ✓ Longer audio provides more temporal context for the model
   ✓ HuBERT can capture longer-range dependencies and patterns
   ✓ Words are too short to exhibit full accent characteristics

3. COARTICULATION EFFECTS:
   ✓ Sentences show how sounds blend across word boundaries
   ✓ These cross-word effects reveal accent-specific articulation
   ✓ Isolated words miss these natural speech patterns

4. FEATURE RICHNESS:
   ✓ 3-5 seconds of speech provides richer feature representation
   ✓ More frames → more robust statistics after pooling
   ✓ Short segments may contain noise or incomplete phonemes

5. ACCENT MANIFESTATION:
   ✓ Accents are most apparent in connected speech
   ✓ Sentence-level captures speaking style and fluency
   ✓ Word-level may only show phonetic variations

WHEN WORD-LEVEL MIGHT BE USEFUL:

Despite lower accuracy, word-level analysis has advantages:
  ✓ Enables fine-grained phonetic analysis
  ✓ Useful for identifying specific accent markers
  ✓ Can process shorter audio clips in real-time applications
  ✓ Allows word-specific accent detection
  ✓ More samples for training (if data is limited)
